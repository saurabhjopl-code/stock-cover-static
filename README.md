# Stock Cover Static - Branded Version

This is the compact, branded static HTML version for **Flipkart FBF Allocation and Stock Analysis** for Rajnandini Fashion India Limited.

Files:
- index.html (branded header, compact UI)
- styles.css (compact styles + responsive)
- script.js (client-side logic; "Live on Website" renamed to "FBF Stock", default SKU A->Z sort, warehouse filter)
- assets/logo.png (placeholder logo)

How to use:
1. Upload index.html, script.js, styles.css, and assets/logo.png to your GitHub repo.
2. Enable GitHub Pages to host (Settings → Pages → main branch → root).
3. Open the published URL and test uploading CSVs.
